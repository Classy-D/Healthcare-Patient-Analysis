### Exploratory Analysis (SQL)
1. Total Revenue Generated ->> Shows the total revenue generated by the healthcare system.
```sql
SELECT 
    SUM(treatment_cost) AS total_revenue
FROM healthcare_patient_analytics;
```
2. Number of Patients by Age Group ->> Identifies the most affected age group.
```sql
SELECT 
    age_group,
    COUNT(patient_id) AS total_patients
FROM healthcare_patient_analytics
GROUP BY age_group
ORDER BY total_patients DESC;
```
3. Patient Distribution by Region ->> Shows regions with the highest healthcare burden.
```sql
SELECT 
    region,
    COUNT(patient_id) AS patient_count
FROM healthcare_patient_analytics
GROUP BY region
ORDER BY patient_count DESC;
```
4. Average Length of Stay by Department ->> Identify departments with longer hospitalization periods.
```sql
SELECT 
    department,
    AVG(length_of_stay_days) AS avg_stay_days
FROM healthcare_patient_analytics
GROUP BY department
ORDER BY avg_stay_days DESC;
```
5. Revenue Generated by Department ->> Shows which departments generate the most revenue.
```sql
SELECT 
    department,
    SUM(treatment_cost) AS department_revenue
FROM healthcare_patient_analytics
GROUP BY department
ORDER BY department_revenue DESC;
```
6. Monthly Revenue Trend ->> Tracks monthly revenue patterns.
```sql
SELECT 
    MONTH(date) AS month_number,
    SUM(treatment_cost) AS monthly_revenue
FROM healthcare_patient_analytics
GROUP BY MONTH(date)
ORDER BY month_number;
```
7. Recovery Performance ->> Shows which departments achieve the best patient recovery outcomes.
```sql
SELECT 
    department,
    ROUND(AVG(recovery_score),2) AS avg_recovery_score
FROM healthcare_data
GROUP BY department
ORDER BY avg_recovery_score DESC;
```

### Recommendations
1. 🧠 Shift from Treatment → Prevention
- Increase investment in:
- Regular health screenings
- Early diagnosis programs
- Community health campaigns

2. 👨‍💼 Target the 31–45 Age Group
- Workplace wellness programs
- Mental health support
- Fitness awareness campaigns
 This will:
- Reduce hospital visits
- Improve productivity

3. 🌍 Focus on High-Burden Regions (West & North)
- Build/upgrade healthcare facilities in these regions
- Deploy mobile clinics
- Improve sanitation & environmental health

4. 🛏️ Reduce Length of Hospital Stay
- Improve early diagnosis
- Use outpatient care where possible
- Invest in rehabilitation and home care

5. 🧘 Promote Therapy & Rehabilitation
- Encourage post-treatment therapy
- Subsidize physiotherapy and rehab services
  Which:
- Prevent relapse
- Reduce future hospital admissions

6. 📅 Prepare for Seasonal Demand ->> Revenue fluctuations across months
- Identify diseases linked to months (e.g., malaria, flu)
- Stock resources in advance
- Run seasonal awareness campaigns
